# ADC Example

This example demonstrates the function of ADC to measure input voltage.
The example will periodically print the measured voltage values.

Hardware configuration:
According to the pin index information of the development board, connect the voltage signal to be measured to the corresponding ADC pin.
[Board Pin Infos](https://github.com/hpmicro/arduino/blob/main/variants)

```
PIN_TEST: 3.286\n.
```
